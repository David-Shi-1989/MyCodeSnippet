# Whistle

## 安装

```bash
npm install -g whistle
```

## 启动

```bash
w2 start
```

![image-20220114144519157](https://raw.githubusercontent.com/David-Shi-1989/img-bed/master/20220114144519.png)

默认端口为`8899`，在浏览器输入`http://127.0.0.1:8899`即可打开界面。

### 重启

```bash
w2 restart
```

### 关闭

```bash
w2 stop
```
